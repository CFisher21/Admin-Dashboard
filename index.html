Etch-a-sketch

The "Etch-a-Sketch" web application replicates the classic drawing toy experience digitally. The webpage features a central grid area where users can draw by hovering over grid tiles, which change color either to black or randomly when "Rainbow Mode" is activated. The interface is clean and responsive, designed with flexbox for layout and animations for visual effects, ensuring an engaging and nostalgic drawing experience reminiscent of the original Etch-a-Sketch toy.
